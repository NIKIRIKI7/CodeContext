# CodeContext AI

![CodeContext AI](assets/images/logo.png)

## О проекте (About)

**CodeContext AI** — это мощный инструмент для подготовки кодовой базы к работе с большими языковыми моделями (LLM), такими как ChatGPT, Claude, Cursor и другими.

**Почему был создан продукт?**
При работе с нейросетями разработчики постоянно сталкиваются с ограничениями размера контекстного окна (Token Limit) и тем, что ИИ "теряет" архитектурную связность проекта при копировании кода частями. CodeContext AI создан, чтобы решить эту проблему, позволив в пару кликов собрать весь проект в один идеально структурированный промпт.

**Для чего это нужно и какие проблемы решает?**
Инструмент анализирует выбранные папки, учитывает `.gitignore`, вырезает мусор (комментарии, пустые строки) и генерирует единый текстовый файл. С помощью функции **"Skeleton ☠️"** можно даже удалить реализацию функций, оставив только их названия и структуру классов — это позволяет нейросети "понять" архитектуру гигантского проекта, потратив минимум токенов.

**Чем CodeContext AI лучше аналогов?**
- **Умное сжатие (Minify & Skeleton):** Сокращает потребление токенов до 80%.
- **LLM Patcher:** Уникальная функция применения патчей. Нейросеть возвращает JSON с изменениями, а программа сама применяет их к нужным файлам на диске.
- **LLM Checker:** Встроенная автоматическая проверка предложенного нейросетью кода (через API) на наличие синтаксических ошибок перед его сохранением.
- **Глубокий анализ зависимостей (AST/Regex):** Понимает Python, JS/TS, Vue и другие языки, строя карту импортов.
- **Интеграция с ОС:** Добавляется прямо в контекстное меню Windows/Linux.

## 🚀 Основные возможности

* **Drag & Drop UI:** Удобный графический интерфейс на PySide6 с поддержкой тем (Apple, Modern).
* **CLI-режим:** Возможность использования инструмента из консоли или CI/CD пайплайнов.
* **Поддержка Git:** Автоматическое исключение файлов из `.gitignore` и фильтрация только измененных файлов.
* **Кастомные шаблоны (Jinja2):** Вывод результата в Markdown, XML, Plain Text или через ваши собственные шаблоны (11 встроенных).
* **Интерактивный Diff-просмотр:** Безопасный предпросмотр изменений кода до и после оптимизации.
* **Локальный подсчет токенов:** Использует `tiktoken` (алгоритмы OpenAI) для точного прогнозирования затрат контекста.

## 🛠 Установка (Getting Started)

### Предварительные требования
* Python 3.10 или выше.
* Установленный `git` (если планируется клонирование репозиториев прямо из приложения).

### Инструкция по установке из исходников

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/NIKIRIKI7/CodeContext.git
   cd codecontext-ai
   ```

2. Создайте виртуальное окружение и активируйте его:
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   # Для Linux/macOS:
   source venv/bin/activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## 💻 Использование (Usage)

### Запуск графического интерфейса (GUI)
Просто запустите главный файл проекта:
```bash
python main.py
```
- Перетащите папку с вашим проектом в окно программы.
- Выберите нужные файлы в дереве (ненужное отфильтруется по `.gitignore`).
- Настройте фильтры (Minify, Skeleton).
- Нажмите **"📋 В Буфер обмена"** и вставьте результат в ChatGPT/Claude.

### Использование через командную строку (CLI)
CodeContext AI отлично работает в терминале:
```bash
# Базовое сканирование папки
python main.py --cli --path /путь/к/вашему/проекту

# Запуск с определенным режимом сбора зависимостей (default, shallow, deep)
python main.py --cli --path /путь/к/проекту --mode deep

# Автоматическое применение JSON-патча от нейросети
python main.py --cli --path /путь/к/проекту --patch /путь/к/ответу_llm.json
```

## 🏗 Технологии (Tech Stack)

* **Язык:** Python 3.10+
* **GUI Фреймворк:** PySide6 (Qt)
* **Архитектура:** Clean Architecture + Redux-паттерн для управления состоянием (Unidirectional Data Flow).
* **Ключевые библиотеки:** `asyncio` (асинхронные I/O операции), `tiktoken` (подсчет токенов), `jinja2` (шаблонизация).
